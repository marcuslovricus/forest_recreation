Here I assess the forest recreation patterns in Europe
Original data is from Europe-wide survey from CLEARING-HOUSE project. Independent variables are all GIS data on Europe's forests that I could find, and also location embeddings from satellite images 
All the data is on 1 x 1 km spatial resulation
Data is available on Zenodo at https://zenodo.org/records/19163771. Use version from March 2026, you can find the metadata / explanation of data there as well.
